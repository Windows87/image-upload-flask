## How to Run
1. Start MySQL
2. Run:
```bash
# Clone this repository
git clone https://github.com/Windows87/image-upload-flask/
# Go into repository
cd image-upload-flas
# Install dependencies
pip install -r requirements.txt
# Start
python run.py
```
3. Go to http://localhost:5000
